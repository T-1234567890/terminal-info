Website Privacy

What this website does

This website is a static product site for Terminal Info. Its purpose is to explain the project, link to documentation, and point visitors to official download and source locations.

Analytics

This website uses Cloudflare Web Analytics to understand general traffic patterns, such as page views and visits.

This analytics system is privacy-focused and does not use cookies or track personal identity.

External links

The website links out to external services such as GitHub, crates.io, and lib.rs.

Those services operate under their own terms and privacy policies once you leave this site.

CLI privacy is separate

Using this website is not the same as running the Terminal Info CLI. The CLI runs locally on your machine and has its own separate policy page.
